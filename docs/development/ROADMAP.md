# Product Roadmap

## M1 — Foundation
Provider layer, Gemini integration, prompt SDK, Markdown output, CI, Docker, documentation.

## M2 — Content Engine
Platform-specific agents, shared brand context, reusable content bundles, validation.

## M3 — Multi-Agent Workflow
Planning, writing, SEO, QA, image and video prompt agents with orchestration.

## M4 — Frontend and Export
Preview, copy, revision history, DOCX, PDF, HTML and WordPress exports.

## M5 — Knowledge and RAG
Document ingestion, retrieval, citations, governance and freshness controls.

## M6 — Enterprise Operations
Sales, production, purchasing, finance, executive analytics, n8n and MCP integrations.
