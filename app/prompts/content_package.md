ผลลัพธ์หลักต้องเป็น “สคริปต์คอนเทนต์” ไม่ใช่ JSON และต้องจัดเป็น Markdown ตามลำดับนี้:

# บทความเว็บไซต์
เขียนบทความที่อ่านลื่น มีหัวข้อย่อย ความรู้ด้านการผลิต เคล็ดลับจากโรงงาน บทสรุป และ FAQ 5 ข้อ

# Facebook
เขียนโพสต์พร้อมใช้ มี Hook เนื้อหา คุณค่าที่ลูกค้าได้รับ CTA ข้อมูลติดต่อ และแฮชแท็ก

# Instagram
เขียนแคปชันพร้อมใช้ กระชับ อ่านง่าย มี CTA และแฮชแท็ก

# X
เขียนโพสต์ไม่เกินข้อจำกัดทั่วไปของแพลตฟอร์ม กระชับและไม่ยัดแฮชแท็กมากเกินไป

# LinkedIn
เขียนโพสต์เชิงธุรกิจสำหรับเจ้าของกิจการ ฝ่ายจัดซื้อ และองค์กร

# SEO
ระบุ Primary keyword, Secondary keywords, Long-tail keywords, Meta title, Meta description และ Slug เป็นข้อความอ่านง่าย

# Image Prompt
เขียน Prompt ภาษาอังกฤษสำหรับภาพ photorealistic แบบ modern corporate แสดงวัสดุกระเป๋าสมจริง แสงธรรมชาติ องค์ประกอบสะอาด ไม่มีข้อความ ไม่มีโลโก้ และไม่มีลายน้ำ
